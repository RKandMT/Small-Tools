## 5. ANSYS数据导出
ANSYS中的数据（如节点号信息、位移计算结果、应力计算结果等）大部分都不能直接复制粘贴出来，必须要通过导出的方式才能取出至其他文件中。

导出数据的思路如下：
1. 将需要导出的数据\*get到（数组）变量中；
2. 用\*VWRITE或\*MWRITE将（数组）变量值写入txt；
3. txt可以用excel打开，对打开的表格进行固定宽度的分组即可得到需要的数组。
