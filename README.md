## 5. 用python求解齐次线性方程组
形如Ax=0，已知A，求解x的过程称为求解齐次线性方程组，计算思路如下：
1. 先判定解的性质：对于n列的矩阵A，若它的秩rank(A)=n，则仅有零解；若它的秩rank(A)<n，则除了零解之外，还有非零解；
2. 利用scipy库自带的化零矩阵进行求解；
3. 输出的结果为全部的特解（也可以更改代码使其只输出其中的一个），以及解带回方程验算的结果（通常不等于0，但会无限趋近0）。
